# OpenJDK docker images

[![Slack](https://philips-software-slackin.now.sh/badge.svg)](https://philips-software-slackin.now.sh)

This repo will contain the openJDK docker images

Current versions available:
```

├── 10
│   ├── jdk
│   │   └── slim
│   │       └── Dockerfile
│   └── jre
│       └── slim
│           └── Dockerfile
├── 8
│   └── jre
│       └── alpine
│           └── Dockerfile
```
